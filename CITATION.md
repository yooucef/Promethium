# Citation

If you use Promethium in your research or production work, please cite it using the following BibTeX entry:

```bibtex
@software{promethium2025,
  author = {Promethium Contributors},
  title = {Promethium: Advanced Seismic Data Recovery and Reconstruction Framework},
  year = {2025},
  url = {https://github.com/olaflaitinen/Promethium},
  version = {2.0.0}
}
```

We also recommend citing the specific versions used for reproducibility.
